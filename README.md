# LinkedIn Weekly Post Automation

A tool to generate and post weekly LinkedIn updates showcasing interactive webapps from [Linguistic Communication](https://linguistic-communication.com/webiles/).

## What it does

Every week, a random webapp from our collection of 28+ interactive learning tools is featured in a LinkedIn post on the [Linguistic Communication LinkedIn profile](https://www.linkedin.com/in/linguisticcommunication/).

The posts promote the webapps as proof of concept for schools interested in custom interactive learning tools.

## Live UI

The LinkedIn Poster tool is available at:
https://linguistic-communication.com/automation/linkedin-poster.html

## Auto-deployment

Any changes pushed to `main` are automatically deployed to Hostinger via GitHub Actions (FTP).

- **Digital Technologies** (10 apps) — hardware, networking, cybersecurity, OS
- **Agile** (4 apps) — Scrum, user stories, acceptance testing
- **Artificial Intelligence** (4 apps) — prompt engineering, AI strategy
- **Software Development** (3 apps) — HTML, CSS, UAT
- **Management** (3 apps) — system design, security audits, enterprise alignment
- **Communication** (4 apps) — presentations, team roles, stakeholder management
